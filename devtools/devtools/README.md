# `devtools/devtools/` (Python devtools)

Python scripts used for development and “prepublish” sanity checks.

Notable scripts:
- `prepublish_smoke.py`: smoke checks (imports, JSON validity, basic sanity)
- `build_filtered_pds_catalog.py`: helper to build filtered catalogs
- `download_pds_one_per_camera.py`: helper downloader for quick samples
