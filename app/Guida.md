# Cos'e questa app
MSL Image Downloader ti aiuta a trovare, filtrare, scaricare e processare immagini della missione Mars Science Laboratory.

Puoi usare l'app per:
- cercare immagini per `sol`
- scegliere una o piu `camere`
- scaricare i file
- processare le immagini
- organizzare i risultati nella cartella di output

# Come iniziare
1. Inserisci il tuo nome e accedi.
2. Apri `Configurazioni`.
3. Scegli lingua e tema.
4. Imposta la cartella di download.

Nota:
Il `download path` e necessario se vuoi salvare o processare file.

# Come trovare immagini
1. Imposta `Sol start` e `Sol end`.
2. Seleziona una o piu camere.
3. Scegli la sorgente:
- `PDS`
- `RAW archive`
4. Se vuoi, imposta:
- `Min img size`
- `Max images`
5. Premi `Apply filters`.

Dopo questo passaggio vedrai quante immagini corrispondono ai filtri scelti.

# Come eseguire un'azione
Dopo aver applicato i filtri, premi `RUN` per eseguire il workflow completo:
- `Download + Process + Organize`

Se vuoi interrompere un'operazione in corso, premi `Stop`.

# Come controllare i risultati
Durante e dopo l'operazione puoi controllare:
- `OUTPUT` per messaggi e stato
- `Viewport Immagine` per l'anteprima
- `Metadati` per i dettagli tecnici
- `Selected Images` per i file gia salvati

# Esempi semplici
- Trova immagini `NAVCAM` tra `sol 720` e `750`
- Scarica solo `10` immagini `MASTCAM`
- Scarica, processa e organizza le immagini filtrate

# Suggerimenti utili
- Parti con range piccoli per fare test veloci
- Controlla sempre i filtri prima di lanciare l'azione
- Verifica la cartella di output prima di scaricare
- Usa `Max images` se vuoi fare prove rapide

# Cosa aspettarti
Alcune operazioni possono richiedere tempo, soprattutto con molte immagini o con processing attivo.

Se una ricerca restituisce troppi risultati, restringi i filtri e riprova.

# What this app is
MSL Image Downloader helps you find, filter, download, and process images from the Mars Science Laboratory mission.

You can use the app to:
- search images by `sol`
- choose one or more `cameras`
- download files
- process images
- organize results in the output folder

# How to start
1. Enter your name and log in.
2. Open `Configurations`.
3. Choose language and theme.
4. Set the download folder.

Note:
The `download path` is required if you want to save or process files.

# How to find images
1. Set `Sol start` and `Sol end`.
2. Select one or more cameras.
3. Choose the source:
- `PDS`
- `RAW archive`
4. If needed, set:
- `Min img size`
- `Max images`
5. Press `Apply filters`.

After this step you will see how many images match your filters.

# How to run an action
After applying filters, press `RUN` to execute the full workflow:
- `Download + Process + Organize`

If you want to stop a running operation, press `Stop`.

# How to check results
During and after the operation you can check:
- `OUTPUT` for messages and status
- `Image Viewport` for preview
- `Metadata` for technical details
- `Selected Images` for files already saved

# Simple examples
- Find `NAVCAM` images between `sol 720` and `750`
- Download only `10` `MASTCAM` images
- Download, process, and organize the filtered images

# Useful tips
- Start with small ranges for quick tests
- Always check filters before running the action
- Verify the output folder before downloading
- Use `Max images` for quick trials

# What to expect
Some operations can take time, especially with many images or with processing enabled.

If a search returns too many results, narrow the filters and try again.
